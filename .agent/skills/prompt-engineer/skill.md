# Instructions
1. Khi người dùng muốn trích xuất một loại tài liệu mới (vd: Bằng lái xe).
2. Hãy phân tích các trường thông tin cần lấy.
3. Tạo ra một Prompt yêu cầu LLM trích xuất đúng định dạng JSON, có xử lý trường hợp dữ liệu bị mờ/hỏng.
