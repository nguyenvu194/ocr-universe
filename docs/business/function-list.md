## Trường hợp sử dụng (Use Cases)

| # | Use Case | Mô tả |
|---|---|---|
| 1 | **Trích xuất dữ liệu** | Hóa đơn, biên lai, biểu mẫu, bảng biểu → database / spreadsheet |
| 2 | **Hỗ trợ tiếp cận (Accessibility)** | Chuyển tài liệu in/viết tay → text để đọc bằng screen reader |
| 3 | **Số hóa sách** | Sách in → văn bản số dễ tìm kiếm và phân phối |
| 4 | **Chỉnh sửa văn bản** | Chuyển đổi ảnh → text để chỉnh sửa nội dung |
| 5 | **Phân tích dữ liệu** | Trích xuất text từ ảnh để phân tích thị trường, nội dung |
| 6 | **Giáo dục** | Chuyển ghi chú viết tay, sách giáo khoa → text dễ tổ chức |
| 7 | **Pháp lý & tuân thủ** | Trích xuất thông tin từ hợp đồng, tài liệu chính phủ |
| 8 | **Tự động hóa doanh nghiệp** | Thay thế nhập liệu thủ công, tối ưu hóa quy trình hành chính |

## Đối tượng người dùng

| # | Đối tượng | Mô tả |
|---|---|---|
| 1 | **Văn phòng** | Xử lý hóa đơn, biên lai, nhập liệu, quản lý database |
| 2 | **Cá nhân** | Số hóa ghi chú, chuyển sách in → ebook, trích xuất text từ screenshot |
| 3 | **Số hóa tài liệu** | Chuyển giấy tờ in → bản số dễ tìm kiếm, chỉnh sửa, chia sẻ |
| 4 | **Sinh viên** | Chuyển ghi chú lớp học viết tay → text số |
| 5 | **Báo chí / Truyền thông** | Trích xuất nội dung báo giấy → chia sẻ lên mạng xã hội |
